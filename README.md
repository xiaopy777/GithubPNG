# GithubPNG

Personal image hosting repository.

## Usage

Put images under `images/`, then use this raw URL pattern:

```text
https://raw.githubusercontent.com/xiaopy777/GithubPNG/main/images/your-image.png
```

Markdown example:

```markdown
![description](https://raw.githubusercontent.com/xiaopy777/GithubPNG/main/images/your-image.png)
```

Keep file names simple, for example `2026-06-08-screenshot.png`, so links stay readable.
